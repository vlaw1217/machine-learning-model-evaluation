# Machine Learning Model Evaluation

This project demonstrates a machine learning text classification workflow using Python and scikit-learn. The notebook applies text preprocessing, Bag-of-Words, TF-IDF feature extraction, Logistic Regression, train-test splitting, confusion matrix evaluation, and prediction comparison.

## Project Overview

The goal of this project is to classify text data using traditional machine learning techniques. The workflow includes preparing text data, converting text into numerical features, training a classification model, and evaluating prediction performance.

## Workflow

The notebook follows these main steps:

1. Load the dataset
2. Explore the dataset structure
3. Preprocess text data
4. Convert text into numerical features using Bag-of-Words
5. Convert text into numerical features using TF-IDF
6. Split the dataset into training and testing sets
7. Train a Logistic Regression model
8. Make predictions on test data
9. Evaluate the model using a confusion matrix
10. Compare actual labels with predicted labels
